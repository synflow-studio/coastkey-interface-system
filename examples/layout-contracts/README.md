# Layout Contract Examples

These examples are simplified public-safe snippets. They are not full product
source and are not intended to run as a standalone app.

They show:

- a slot-based Product Page Layout contract;
- left/main/right composition;
- runtime assembly passing slot content;
- layout ownership separated from runtime state.

Files:

- `ProductVendorSectionPageLayout.example.tsx`
- `RuntimeSlotsAssembly.example.tsx`
