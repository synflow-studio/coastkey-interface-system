# Task Brief Example

## Goal

Create a source-backed Product Page Layout for a vendor operational section.

## Layer

Product Page Layout.

## Owner Route

Use the product package layout owner. Runtime adapters provide data, callbacks,
and selected state.

## Boundaries

- Do not create reusable visible units as page-local JSX.
- Do not import source data into product components.
- Do not change unrelated runtime behavior.
- Add reference coverage when the layout becomes source-backed.

## Validation

- Typecheck the affected package/app.
- Review reference metadata.
- Run responsive visual QA before public showcase capture.
